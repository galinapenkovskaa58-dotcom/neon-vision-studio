
# Сайт нейрофотографа — план реализации

## Дизайн-система
- **Тёмная тема**: фон `#070A12`, неоновые акценты `#2D7CFF`, `#37D9FF`, `#B04BFF`, `#FF4FD8`
- **Glass-морфизм**: полупрозрачные карточки с `backdrop-blur`, неоновые border-glow
- **Градиенты** между акцентными цветами, скруглённые элементы (`border-radius: 1rem+`)
- **Анимации**: плавные fade/slide при скролле, glow-пульсация на кнопках и акцентных элементах

## Бэкенд (Lovable Cloud / Supabase)
- **Таблица `bookings`**: имя, телефон, мессенджер (Telegram/WhatsApp/другое), выбранный тариф, стиль, референсы (текст + ссылки), срочность, статус заявки, дата создания
- **Таблица `portfolio`**: заголовок, описание, стиль/категория, URL изображения (Supabase Storage), порядок сортировки
- **Таблица `tariffs`**: название, описание, цена, список включённого, порядок сортировки, активность
- **Таблица `reviews`**: имя клиента, текст отзыва, оценка, фото (опционально)
- **Auth**: email + пароль для входа в админку
- **RLS**: публичные SELECT для portfolio/tariffs/reviews, INSERT для bookings; полный доступ для admin-роли
- **Storage bucket** для загрузки фотографий портфолио

## Главная страница (лендинг)

### 1. Header
- Логотип/название, навигация по секциям (якоря), кнопка «Записаться» с glow-эффектом
- Sticky, glass-эффект при скролле

### 2. Hero-секция
- Нестандартная композиция: крупное фото/видео-фон с overlay-частицами, абстрактные нейро-узоры
- Заголовок с градиентным текстом, подзаголовок, CTA-кнопка «Записаться на сессию» с неоновым свечением
- Плавная анимация появления элементов

### 3. Портфолио (примеры работ)
- Динамическая галерея, подгружаемая из БД
- Фильтрация по стилям, hover с glow-рамкой и раскрытием деталей
- Lightbox для просмотра в полном размере

### 4. Стили съёмок
- Карточки с glass-эффектом, каждый стиль — уникальная цветовая подсветка
- Описание + мини-примеры

### 5. Тарифы
- Карточки тарифов из БД, glass + градиентные бордеры
- Кнопка «Выбрать» ведёт к форме записи с предзаполненным тарифом

### 6. Отзывы
- Карусель/слайдер с отзывами из БД, glass-карточки

### 7. Форма записи
- Поля: имя, телефон, мессенджер (выбор), тариф (выпадающий из БД), желаемый стиль, референсы (текст), срочность (обычная/срочная)
- Валидация, отправка в Supabase, анимированное подтверждение

### 8. Footer
- Контакты, ссылки на соцсети, ссылки на оферту и политику конфиденциальности

## Дополнительные страницы
- **Политика конфиденциальности** (`/privacy`)
- **Оферта** (`/offer`)

## Админ-панель (`/admin`)
- **Авторизация**: email + пароль через Supabase Auth, защита по роли admin
- **Заявки**: таблица с фильтрами по статусу (новая/в работе/завершена), просмотр деталей, смена статуса
- **Портфолио**: добавление/редактирование/удаление работ, загрузка фото, сортировка
- **Тарифы**: CRUD для тарифов — название, цена, описание, вкл/выкл
- **Отзывы**: добавление/редактирование/удаление

## Этапы реализации
1. Настройка дизайн-системы (CSS-переменные, утилиты glass/glow)
2. Создание таблиц и Storage в Supabase, настройка RLS
3. Лендинг: header, hero, портфолио, стили, тарифы, отзывы, форма, footer
4. Страницы оферты и политики
5. Админ-панель: авторизация, управление заявками, портфолио, тарифами, отзывами

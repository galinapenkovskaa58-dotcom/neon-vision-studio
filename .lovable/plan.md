

# Создание админ-пользователя

## Что нужно сделать
1. Создать edge function `create-admin`, которая:
   - Создаёт пользователя `admin@admin.com` с паролем `admin123` через Supabase Admin API
   - Добавляет запись в `user_roles` с ролью `admin`
   - Включает auto-confirm, чтобы не требовалось подтверждение email

2. Вызвать эту функцию один раз для создания пользователя

3. Включить auto-confirm для email-регистрации (чтобы админ мог сразу войти)

## Результат
После выполнения можно будет войти в `/admin` с логином `admin@admin.com` и паролем `admin123`.

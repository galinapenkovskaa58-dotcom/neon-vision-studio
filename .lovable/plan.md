
## Правка формы капсулы — прямоугольная вертикальная

Меняю только форму летящей капсулы. Вся остальная логика (полёт по линии, морфинг иконок Лампочка → Перо → Искры → Галочка, смена цвета по сегментам, цикл с паузой и сбросом) — без изменений.

## Капсула

- Размеры: `w-10 h-14` (40×56) — выраженно вертикальный прямоугольник.
- Скругление: `rounded-2xl` (мягкие углы, не круг).
- Стиль: `glass-strong`, бордер 2px в цвете текущего сегмента, неоновый `box-shadow` в том же цвете.
- Иконка внутри: `w-5 h-5`, центр через `flex items-center justify-center`, с `drop-shadow` под цвет.
- Halo вокруг: `absolute inset-[-8px] rounded-2xl blur-xl` с radial-gradient в цвет сегмента — повторяет прямоугольную форму капсулы.
- Под капсулой — мягкий «хвост»-glow, чтобы сохранить ощущение полёта.

## Что НЕ меняем
- Файл `src/components/landing/Process.tsx` — единственное место правок.
- Логика framer-motion: `animate={{ left: '12.5% | 37.5% | 62.5% | 87.5%' }}`, `duration: 1.5, ease: 'easeInOut'`.
- Морфинг иконки через `AnimatePresence mode="wait"`, переход 0.25с (scale + rotate).
- Палитра по сегментам: cyan → blue → purple → pink.
- Цикл 0→1→2→3 → пауза ~2с → мгновенный сброс.
- Бегущий блик на линии-коннекторе, мини-«встреча» активной карточки (scale 1.05 + усиленный glow).
- `useReducedMotion()` → статичный режим.
- Mobile (`<md`): анимация скрыта, карточки с поочерёдным glow-pulse.
- Карточки шагов, тексты, иконки, grid и адаптив — без изменений.
- Остальные секции страницы — без изменений.
